## 主要功能

- **无限画布**：自由拖拽节点，搭建专属 AI 工作流
- **文本生成**：支持多模型对话和内容生成
- **图片生成**：支持文生图、图生图，异步任务自动轮询
- **视频生成**：支持文生视频、图生视频，长任务自动等待
- **ComfyUI 接入**：支持本地 ComfyUI 工作流节点
- **节点联动**：框选多节点批量执行，上下游结果自动传递
- **消费记录**：实时查看账户余额和消费明细

## 系统要求

- 操作系统：Windows 10 / 11（64位）
- 需要星沃账号（注册后充值即可使用）

## 安装步骤

1. 下载 `星沃AI1.0 Setup x.x.x.exe`
2. 双击运行安装程序，按提示选择安装目录完成安装
3. 打开软件，注册或登录星沃账号
4. 充值后即可使用文本、图片、视频生成功能
5. 如需使用 ComfyUI 节点，在右上角设置中填写本地 ComfyUI 地址
